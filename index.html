HTML

  • Head elements define elements of Metadata
  • Body elements are items that the user actually interacts with
  • HTML itself is the skeleton of the website, explaining how the website is structured/organized.

CSS

  • A margin indicates how much space we want around the outside of the element.
  • A padding indicates how much space we want around the content inside an element
  • Inline CSS - style attributed to a single element of HTML ~ less flexible than other forms of CSS

GIT

  • git status - Tells us which file we're in
  • git checkout "" - Moves you to a different folder

JavaScript

  • A variable is a named container that allows us to store data in our code
  • Control flow is the order in which a cmputer executes code in a script
  • A function is simply a set of instructions that tells the computer how to perform a certain task
  • Array is a single variable that can hold a group of data
